# 日本麻將遊戲

單機日本麻將（Riichi Mahjong）。純 HTML+JS+CSS，無任何構建工具或外部依賴。

線上遊玩：<https://buffalobill-taiwan.github.io/jpmj/>

## 快速開始

```bash
python3 -m http.server 8080
open http://localhost:8080
```

## 功能

- 136 張牌（無花牌、無赤牌）
- Unicode U+1F000..U+1F02F 渲染牌面（使用 [Riichi-Mahjong 字型](https://github.com/rutopio/Japanese-Mahjong-Font-with-OpenType/)）
- 完整的標準役種支援：
    - **一般役**：立直、一發、**雙立直**、門前清自摸和、平和、斷幺九、一盃口、二盃口、役牌、**嶺上開花**、**槍槓**、海底、河底、三色同順/同刻、一氣通貫、混全帯么九、純全帯么九、對々和、三暗刻、混老頭、小三元、三槓子、七對子、混一色、清一色
    - **役滿**：天和、地和、**國士無雙**、**大三元**、**四暗刻**、**字一色**、**綠一色**、**清老頭**、**九蓮寶燈**、**四槓子**、**大四喜**、**小四喜**
- 中途流局（九種九牌、四風連打、四槓散了、四家立直、三家和）及荒牌流局（不聽罰符）
- 連莊／輪莊處理
- **分支規則處理**：
    - **頭跳 (Atama-hane)**：多家榮和時，僅由順位在前者胡牌。
    - **三家和了流局 (Sancha-ron)**：三家同時榮和時，該局以流局處理。
    - **沒有人和 (No Renhou)**：不支援「人和」役種。
- 六種 AI 類型（詳見 [AI 玩家](#ai-玩家) 章節）。
- 託管模式：可獨立為每家設定 AI 類型，包含人類玩家在內皆可切換。
- 對戰長度三種選擇：東風戰（4 局）、半莊戰（8 局）、一莊戰（16 局），最後一局標示 **All Last**。
- 起始座位選擇（隨機／東／南／西／北）
- 牌面風格切換（彩色／單色）
- 聽牌提示
- 視覺優化：立直宣告牌自動橫置，穩定的捨牌區佈局。

## AI 玩家

| AI 類型 | 特徵 | 目標役種 | 鳴牌策略 |
|---------|------|---------|---------|
| **初學者 (Beginner)** | 新手級，簡單直覺 | **斷幺九**、**役牌** | 有助於斷幺九/役牌且向聽數倒退就鳴；無特定目標時隨便鳴 |
| **一般人 (Normal)** | 中等難度，目標多樣 | **立直**、**斷幺九**、**混/清一色**、**對對和** | 向聽數減少為前提，參考目標役種篩選 |
| **高手 (Expert)** | 最強難度，攻防兼備 | 全役種（含 **國士無雙**、**七對子**） | 高級向聽數評估 + 目標篩選；末盤存活模式（有人立直且未聽牌則完全防守） |
| **国士命 (Kokushi)** | 特化型，全力拼國士 | **國士無雙**（唯唯一） | 完全不鳴牌；起手留全部么九牌打其他牌；非役滿不和牌 |
| **断么廚 (Tanyao)** | 特化型，斷幺至上 | **斷幺九**（唯一目標） | 非么九鳴牌正常判斷；若手牌仍有么九則亂鳴（試圖快速清掉）；Chi set 含么九牌則跳過 |
| **門清俠 (Menzen)** | 特化型，門前至上 | **立直**、**門前清自摸和** | 不吃、不碰、不明槓；聽牌後 85% 立直；起手九種九牌時宣告流局 |

所有 AI 共用基底行為：
- **智慧捨牌**：開牌後積極打掉不符合目標役種的牌。
- **向聽數估算**：每捨牌前評估所有可能的捨牌選擇，優先保留有利於聽牌的牌。
- **危險度評估**：高手在存活模式下使用筋牌判斷，其他 AI 僅考慮基本危險度。

## 細微規則

以下為實作中容易誤解的細微規則，記錄以供開發參考：

- **搶槓（槍槓）**：加槓（kakan）時若其他玩家能以該張牌和牌，則**槓牌不成立**——不增加寶牌指示牌、不清除任何人的一發狀態。此行為與大明槓不同。暗槓（ankan）後，若其他玩家（特別是國士無雙）能以該張牌榮和，則觸發搶暗槓規則。
- **呼叫優先序 (Call Priority)**：優先級為 **榮和 > 碰/槓 > 吃**。當多名玩家同時宣告不同呼叫時，系統會自動執行優先序最高者，並在 Log 中記錄攔截資訊。
- **頭跳（Atama-hane）**：多家同時榮和時，僅由座位順位在前者（巡目較早者）和牌，其餘榮和無效。系統會在 Log 提示「頭跳」判定結果。
- **三家和了流局（Sancha-ron）**：若三家同時**宣告**榮和（非僅具備資格），該局以流局處理。
- **人和（Renhou）**：本遊戲不支援人和役種。
- **四槓散了（Suukantsu）**：若累計四次槓且由不同玩家執行，則中途流局。同一人執行四槓時不構成流局（觸發四槓子役滿）。
- **立直限制**：僅在門前清、且牌山剩餘張數 **≥ 4 張** 時可宣告。宣告立直需支付 1000 點，若點數不足則不可立直。
- **立直提示**：點數不足時，畫面上會出現「立直(無点)」按鈕提示玩家。無役榮和時亦顯示「榮和(無役)」提示。
- **平和判定**：平和（ピンフ）成立需滿足所有條件，其中雀頭須為數牌、風牌（非自風非門風）或三元牌以外的牌。**三元牌（中發白）不可作為平和的雀頭**。
- **荒牌流局的立直棒**：立直棒保留至下一局（無論有無聽牌者）。
- **無擊飛規則（No Tobashi）**：玩家分數可降至負值，遊戲不會因此提前結束，���在牌局打滿（All Last 結束）時結算。
