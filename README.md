# 日本麻將遊戲

單機日本麻將（Riichi Mahjong）。純 HTML+JS+CSS，無任何構建工具或外部依賴。

線上遊玩：<https://buffalobill-taiwan.github.io/jpmj/>

## 快速開始

```bash
python3 -m http.server 8080
open http://localhost:8080
```

## 功能

- 136 張牌（無花牌、無赤牌）
- Unicode U+1F000..U+1F02F 渲染牌面（使用 [Riichi-Mahjong 字型](https://github.com/rutopio/Japanese-Mahjong-Font-with-OpenType/)）
- 完整的標準役種支援：
    - **一般役**：立直、一發、**雙立直**、門前清自摸和、平和、斷幺九、一盃口、二盃口、役牌、**嶺上開花**、**槍槓**、海底、河底、三色同順/同刻、一氣通貫、對對和、三暗刻、混全/純全、混老頭、小三元、三槓子、混一色、清一色。
    - **役滿**：天和、地和、**國士無雙**、**大三元**、**四暗刻**、**字一色**、**綠一色**、**清老頭**、**九蓮寶燈**、**四槓子**、**大四喜**、**小四喜**。
- 中途流局（九種九牌、四風連打、四槓散了、四家立直、三家和）及荒牌流局（不聽罰符）
- 連莊／輪莊處理
- **分支規則處理**：
    - **頭跳 (Atama-hane)**：多家榮和時，僅由順位在前者胡牌。
    - **三家和了流局 (Sancha-ron)**：三家同時榮和時，該局以流局處理。
    - **沒有人和 (No Renhou)**：不支援「人和」役種。
- AI 對手，具備「目標役種」意識：
    - **初學者**：專注於 **斷幺九** 與 **役牌**。
    - **一般人**：增加 **混/清一色**、**對對和** 與 **立直** 意識。
    - **高手**：全役種支援，包含 **國士無雙** 與 **七對子** 等極端戰略。具備末盤「存活模式」，若未聽牌且有人立直，會全力棄和防守。
    - **目指し国士**：完全不鳴牌，只留么九牌全力拼 **國士無雙**，非國士無雙絕不和牌。
    - 智慧捨牌：開牌後會積極打掉不符合役種目標的牌。
- 託管模式：可獨立設定 AI 強度，與 CPU 對手相同四種等級。
- 對戰長度三種選擇：東風戰（4 局）、半莊戰（8 局）、一莊戰（16 局），最後一局標示 **All Last**。
- 起始座位選擇（隨機／東／南／西／北）
- 牌面風格切換（彩色／單色）
- 聽牌／防禦提示
- 視覺優化：立直宣告牌自動橫置，穩定的捨牌區佈局。

## 細微規則

以下為實作中容易誤解的細微規則，記錄以供開發參考：

- **搶槓（槍槓）**：加槓（kakan）時若其他玩家能以該張牌和牌，則**槓牌不成立**——不增加寶牌指示牌、不清除任何人的一發狀態。此行為與大明槓不同（大明槓無搶槓，完成後會增加寶牌指示牌並清除一發）。
- **頭跳（Atama-hane）**：多家同時榮和時，僅由座位順位在前者（巡目較早者）和牌，其餘榮和無效。
- **三家和了流局（Sancha-ron）**：若三家同時榮和（包含搶槓形成的三家和），該局以流局處理。
- **人和（Renhou）**：本遊戲不支援人和役種。
- **四槓散了（Suukantsu）**：若累計四次槓且由不同玩家執行，則中途流局。同一人執行四槓時不構成流局（觸發四槓子役滿）。
- **立直後的一發**：一發僅在**成功完成的**吃、碰、明槓（非搶槓）時解除。嶺上開花、榮和、自摸、或未成立的槓（被搶槓）**不解除**一發。
- **荒牌流局的立直棒**：立直棒保留至下一局（無論有無聽牌者）。
